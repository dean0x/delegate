## Architecture & Design

### Key Architectural Decisions

**Event-Driven Layer Separation**:
```
TaskManager.delegate()
    → TaskDelegated event
    → DependencyHandler validates dependencies
    → TaskDependencyFailed event (on error)
    → QueueHandler enqueues task
    → WorkerHandler spawns worker
```

**Repository = Pure Data Layer**:
- No business logic in repositories
- Cycle detection lives in DependencyHandler.validateSingleDependency()
- Repository only does CRUD operations with validation at boundaries

**Test Mode Architecture**:
```
AUTOBEAT_TEST_MODE=true
    → NoOpProcessSpawner (returns MockChildProcess)
    → ResourceMonitor skips startMonitoring()
    → MockChildProcess emits 'exit' immediately
    → Tasks complete without actual spawning
```
