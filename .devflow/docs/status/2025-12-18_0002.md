## Architecture & Design

### Key Architectural Decisions This Session

1. **Two-Phase Handler Initialization**
   - Standard handlers: `EventHandlerRegistry.initialize()` calls `setup(eventBus)` on each
   - DependencyHandler: Factory pattern `create()` for async graph initialization
   - Both returned for unified lifecycle management

2. **Fail-Fast Dependency Extraction**
   - `extractHandlerDependencies()` validates all 10 dependencies upfront
   - Returns immediately on first missing dependency with specific error

3. **Cleanup on Failure**
   - If DependencyHandler creation fails, `registry.shutdown()` called first
   - Ensures no dangling subscriptions on partial initialization
