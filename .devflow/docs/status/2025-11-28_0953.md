### Problems Solved Today

1. **Issue #28 - Graph Corruption (CRITICAL)**
   - **Problem**: `wouldCreateCycle()` used `new Map(this.graph)` which creates shallow copy - Set values are references, not copies
   - **Solution**: Deep copy using `new Map(Array.from(this.graph.entries()).map(([k, v]) => [k, new Set(v)]))`
   - **Tests**: Added 3 regression tests verifying immutability after cycle checks

2. **Spawn Burst Overload**
   - **Problem**: Load average (`os.loadavg()`) is 1-minute rolling average - doesn't reflect sudden resource spikes from recent spawns
   - **Solution**: Implemented "settling workers" tracking - track spawns in 15-second window and project their resource usage
   - **Result**: Prevents fork-bomb scenarios during recovery or burst task submission

3. **Session Crashes During Tests**
   - **Problem**: Concurrent spawns overwhelmed system resources
   - **Solution**: Increased spawn delay from 50ms to 1000ms (tested 50ms → 10000ms → 1000ms)

   - `npm run test:all` - Local terminal/CI only

2. **DependencyHandler uses factory pattern**:
   ```typescript

   // NOT: new DependencyHandler(...)
   ```

3. **Load average is a lagging indicator** - Recent spawns won't appear in `os.loadavg()` for up to a minute. That's why we have settling workers tracking.

4. **Shallow copy gotcha in JavaScript**:
   ```typescript
   // BAD: Set values are references
   new Map(existingMap)

   // GOOD: Deep copy
   new Map(Array.from(existingMap.entries()).map(([k, v]) => [k, new Set(v)]))
   ```
