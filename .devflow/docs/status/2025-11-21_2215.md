## Architecture & Design

### Key Architectural Decisions This Session

**DependencyHandler Factory Pattern:**
```typescript
// Before (risky)
private graph!: DependencyGraph;
constructor(...) { }
async setup(eventBus) { this.graph = ...; }

// After (safe)
private graph: DependencyGraph;
private constructor(..., graph: DependencyGraph) { }
static async create(...): Promise<Result<DependencyHandler>> {
  // Initialize graph, create handler, subscribe to events
  return ok(handler);
}
```
